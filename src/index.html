Google Sheets Access Guide

If a Google Sheet is owned by someone else, how you access it depends entirely on how they shared it:

Scenario 1: Set to "Anyone with the link can view"

If the original owner made the document public via a link, it is extremely easy:

  • You can use the Spreadsheet ID straight from their link.
  • You can use your own API Key (generated in your own Google Cloud Console).
  • The Google API will let your app read it flawlessly without needing any special permissions.

Scenario 2: Shared specifically with your email (Private)

If it is a private document and restricted only to your specific Google Account, a simple API key won't work.

  • What to do: Navigate to Google Cloud Console, find your project, and create a "Service Account".
  • It will give you a robot email address. Tell the owner to "Share" their private Google Sheet explicitly with that robot email address.
  • Under "Keys" in Google Cloud for that Service Account, click "Create New Key" -> JSON. Choose the "Service Account" auth mode in this app, and upload that exact JSON file here. Our built-in secure engine will magically handle the rest!
